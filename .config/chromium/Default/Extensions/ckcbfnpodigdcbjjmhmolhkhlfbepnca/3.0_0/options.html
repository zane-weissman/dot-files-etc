Add Custom Template Shortcuts (requires a little bit of work to setup)

Follow the following instructions to get the necessary values for below:

  1. Go to the Google Docs Template Gallery (https://drive.google.com/templates?view=author or https://drive.google.com/a/your-domain-here.com/templates?view=domain)
  2. View the source of the page and search for <div class="templates-entry-actions">
  3. Right after that <div> is a form that contains all the values you need

Example values: https://drive.google.com/a/mydomain.com/usetemplate?id=1dbDgG31BmisjHwGAL3R7RfzZK7NkvAytnM-9nLxnRZc&category=Letters&mode=domain&type=docs&token=QhGujDcBAAA.jn8mn9GSi-Qt8Hvqcpznag.tZH5cbyx5keCoAFUXoVUPQ).
